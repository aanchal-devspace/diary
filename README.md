# 📖 Personal Diary

A simple Full Stack Personal Diary web application where users can register, log in, and manage their personal diary entries.

## ✨ Features

- 👤 User Registration
- 🔐 User Login
- 📝 Create Diary Entries
- 📚 View All Entries
- ✏️ Edit Entries
- 🗑️ Delete Entries
- 🚪 Logout
- 💾 Data stored using JSON files

---

## 🛠️ Technologies Used

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Node.js
- Express.js

### Storage
- JSON Files (`users.json` & `diary.json`)


---

## 📸 Features Overview

- Register a new account
- Login with existing account
- Add new diary entries
- View all saved entries
- Edit existing entries
- Delete unwanted entries
- Logout securely

---

## 🎨 UI Design

The user interface was designed using custom HTML and CSS.

The color palette and visual theme were selected with the assistance of AI, while the implementation and integration into the project were completed manually.

---

## ⚠️ Note

This project uses JSON files for data storage for learning purposes.

When deployed on platforms such as Render, JSON file data may not persist after server restarts because the platform does not provide persistent local file storage.
